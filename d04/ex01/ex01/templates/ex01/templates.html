Moteur de template Django

Django utilise par défaut le Django Template Language (DTL), un moteur de templates minimaliste et sécurisé qui permet de générer du HTML dynamique à partir de données Python.

1. Les blocs ({% block %})

Les blocs permettent l'héritage de templates (principe DRY). Un template parent définit des zones remplaçables avec {% block nom %}...{% endblock %}. Un template enfant peut surcharger ces zones avec {% extends "parent.html" %} et redéfinir les blocs souhaités. 

— base.html (parent) —
<title>{% block title %}Titre par défaut{% endblock %}</title>
<body>{% block content %}{% endblock %}</body>

— page.html (enfant) —
{% extends "base.html" %}
{% block title %}Ma Page{% endblock %}
{% block content %}<p>Contenu spécifique</p>{% endblock %}

2. Les boucles {% for ... in %}

La balise {% for %} itère sur n'importe quel itérable Python passé dans le contexte. Elle expose aussi des variables spéciales comme forloop.counter (compteur 1-indexé) ou forloop.last. 

{% for item in liste %}
    <li>{{ forloop.counter }}. {{ item }}</li>
{% empty %}
    <p>La liste est vide.</p>
{% endfor %}

3. Les structures de contrôle {% if %}

La balise {% if %} évalue une condition et affiche du contenu conditionnellement. Elle supporte les opérateurs and, or, not, ainsi que les comparaisons (==, !=, <, >, in, not in). 

{% if user.is_authenticated %}
    <p>Bonjour {{ user.username }} !</p>
{% elif user.is_staff %}
    <p>Bonjour administrateur.</p>
{% else %}
    <p>Veuillez vous connecter.</p>
{% endif %}

4. Affichage de variables passées par contexte

La vue passe un dictionnaire context à la fonction render(). Dans le template, les variables du contexte sont accessibles avec la syntaxe {{ variable }}. Les attributs et clés de dictionnaire sont accessibles avec la notation pointée : {{ objet.attribut }}. 

— views.py —
def ma_vue(request):
    context = {
        'titre': 'Bonjour le monde',
        'items': ['Python', 'Django', 'HTML'],
        'user': {'nom': 'Alice'},
    }
    return render(request, 'mon_template.html', context)

— mon_template.html —
<h1>{{ titre }}</h1>
<p>Utilisateur : {{ user.nom }}</p>
{% for item in items %}
    <li>{{ item }}</li>
{% endfor %}

5. Filtres de template

Les filtres modifient la valeur d'une variable : {{ variable|filtre }}. Exemples courants : 

{{ texte|upper }}          → TEXTE EN MAJUSCULES
{{ texte|lower }}          → texte en minuscules
{{ texte|truncatechars:50 }} → texte tronqué à 50 caractères
{{ liste|length }}         → nombre d'éléments
{{ date|date:"d/m/Y" }}    → formatage de date
{{ valeur|default:"N/A" }} → valeur par défaut si vide
